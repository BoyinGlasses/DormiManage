# Ke Hoach Slides: AI Agent Ho Tro Project DormitoryManagement

## Muc Tieu

Trinh bay khai quat AI Agent la gi va cach AI Agent ho tro qua trinh phat trien project DormitoryManagement.

Noi dung tap trung vao vai tro, quy trinh va gia tri cua AI Agent. Khong di sau vao chi tiet tung task, tung dong code hoac qua trinh implement cu the.

## Dinh Huong Thiet Ke Chung

- Phong cach: chuyen nghiep, hien dai, sach, de thuyet trinh.
- Doi tuong nghe: giang vien, ban bao cao do an, nguoi quan tam den ung dung AI trong phat trien phan mem.
- Tong mau chinh: trang, xanh navy, xanh cyan nhe.
- Mau nhan: xanh la hoac vang nhe cho cac diem quan trong.
- Font goi y: Segoe UI, Aptos, Calibri.
- Moi slide nen co mot y chinh ro rang.
- Moi slide chi nen co 4 den 6 dong noi dung.
- Uu tien icon, workflow, module diagram, dashboard mockup.
- Khong nen dung hinh robot qua tre con hoac qua ky ao.
- Khong nen dua anh chup code dai.
- Animation nen nhe: Fade hoac Morph.

## Quy Tac Noi Dung Khi Thuyet Trinh

- Noi o muc khai quat, khong ke lai tung task da lam.
- Tap trung vao cach AI Agent ho tro quy trinh phat trien.
- Khong noi AI tu lam thay con nguoi.
- Khong di sau vao webhook, migration, test case hoac code cu the.
- Co the dung vi du muc cao: hoa don, thanh toan, phan quyen, bao mat, kiem thu.

5. Ket lai bang gia tri: nhanh hon, an toan hon, de kiem soat hon.
